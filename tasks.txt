YouTube から動画を検索・ダウンロードし、
動画再生・音声波形の表示をし、
トリミング・クロップ・メタデータの編集情報を保存・実行できる
electron ツールを作成します。

### 1. 動画再生・基本操作

* [ ] 動画のプレビュー表示（`<video>`タグ）
* [ ] 動画メタデータの取得（再生時間、解像度）
* [ ] 動画の再生/一時停止
* [ ] 再生位置の指定
* [ ] 開始位置へのリプレイ
* [ ] トリミング範囲内でのループ再生

### 2. トリミング機能

* [ ] トリミング開始位置・終了位置の設定（スライダー）
* [ ] トリミング範囲の表示（時間表示）
* [ ] 最小トリミング幅の制限（0.5秒）
* [ ] トリミング範囲の微調整ボタン（±0.05秒、±0.5秒、最小/最大）
* [ ] 現在の再生位置からトリミング位置を設定

### 3. 音声波形・表示

* [ ] WaveSurferによる波形表示
* [ ] 波形のズーム機能（トリミング範囲に応じた自動調整）
* [ ] 波形クリックでの再生位置変更
* [ ] 波形の正規化（Normalize）オプション

### 4. ナビゲーション・UIコンポーネント

* [ ] 再生進行状況バー（VideoProgressBar）
* [ ] クリッピング範囲選択スライダー（VideoClipSlider）
* [ ] 全体の開始位置選択スライダー（VideoStartPositionSlider）
* [ ] スライダーのマーク表示（時間に応じた適切な間隔）
* [ ] クロップ結果のキャンバス表示

### 5. クロップ機能

* [ ] クロップ範囲のプレビュー表示
* [ ] 動画上でのクロップ範囲の可視化
* [ ] 複数のクロップ領域設定（ソース領域と描画領域）
* [ ] クロップ領域の追加/削除
* [ ] 出力サイズ設定（幅・高さ）

### 6. メタデータ編集・入力

* [ ] YouTube Video ID入力
* [ ] ファイル名（Title）入力
* [ ] セリフ（Serif）入力
* [ ] セリフルビ（Ruby）入力
* [ ] カテゴリ選択（複数選択可能、予約済みカテゴリボタン）
* [ ] クリップURL入力（自動生成オプション付き）
* [ ] メモ入力（複数行）

### 7. 自動化・便利機能

* [ ] File Nameの自動入力（Video IDと時間から生成）
* [ ] Nameの定型選択機能
* [ ] ルビ（Ruby）の自動入力（Nameに入力したものを自動でひらがな化）
* [ ] カテゴリ選択肢の設定変更機能

### 8. 外部連携・ダウンロード

* [x] YouTubeの動画検索・ダウンロード機能
* [ ] ダウンロード済みの動画を選択して編集可能にする

### 9. ファイル出力・保存処理

* [ ] 編集後の保存（Saveボタン）でJSONファイル保存
* [ ] 編集後の実行ボタンでMP4ファイル保存（FFmpegコマンド実行）

### 10. UI/UX・その他

* [ ] 編集画面のレイアウト調整（使いやすさの向上）
* [ ] 各入力フィールドのバリデーション
* [ ] キーボードショートカットの実装（Enterキーでの確定など）
